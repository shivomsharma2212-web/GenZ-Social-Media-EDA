

# Gen Z Social Media Usage Analysis

## Overview

This project focuses on **Exploratory Data Analysis (EDA)** of Gen Z social media usage patterns using Python. The goal is to understand user behavior, platform preferences, daily usage habits, addiction levels, and the relationship between social media usage and mental health.

This analysis helps answer important questions such as:

* Which social media platform is most used by Gen Z?
* How many hours do users spend daily on social media?
* Does higher social media usage affect mental health?
* Is there a relationship between age and social media usage?
* What are the addiction level patterns among users?



## Dataset Used

**Dataset Name:** Gen Z Social Media Usage Dataset
**Format:** CSV
**Records:** 1M+ rows

The dataset includes features like:

* Age
* Gender
* Primary Platform
* Daily Usage Hours
* Addiction Level
* Mental Health Score
